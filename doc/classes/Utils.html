    - (String) formatCurrency(value, options) (bound)

    Formats monetary value as a string with decimal and thousands separators

    formatCurrency(1050)
#=> '1.050,00'

formatCurrency(-1050.99, {'decimalSeparator': '.', 'thousandsSeparator': ',', 'absolute': true, 'decimalPlaces': 3}
#=> '1,050.990'

    Parameters:

    • value (Number) — the value to format
    • options (Object)

    Options Hash: (option):

    • decimalSeparator (String) — the character used to separate the decimal and integer parts. Default: ','
    • thousandsSeparator (String) — the character used to separate the thousands. Default: '.'
    • absolute (Boolean) — whether to use an absolute value or not. Default: false
    • decimalPlaces (Integer) — the number of decimal places to use. Default: 2

    Returns:

    • (String) — the value formatted according to the options given

    - (String) pad(str, max, options) (bound)

    Pads a string until it reaches a certain length. Non-strings will be converted.

    pad('19,99', 6)
#=> '019,99'

pad('Hello', 7, {'char': ' ', 'position': 'right'})
#=> 'Hello  '

    Parameters:

    • str (String) — the string to be padded. Any other type will be converted to string
    • max (Integer) — the length desired
    • options (Object)

    Options Hash: (option):

    • char (String) — the character used to pad the string. Default: '0'
    • position (String) — where to pad. Valid: 'left', 'right'. Default: 'left'

    Returns:

    • (String) — the string padded according to the options given

    - (String) getCookieValue(cookie, name) (bound)

    Receives a cookie that has "subcookies" in the format a=b&c=d Returns the content of the "subcookie" with the given name

    c = readCookie('sub')
#=> 'a=b&c=d'
getCookieValue(c, 'a')
#=> 'b'
getCookieValue(c, 'c')
#=> 'd'

    Parameters:

    • cookie (String) — a string with "subcookies" in the format 'a=b&c=d'
    • name (String) — the name of the "subcookie" to get the value of

    Returns:

    • (String) — the content of the "subcookie" with the given name

    - (Object) urlParams() (bound)

    Parses the querystring and returns its object representation. It decodes URI components (such as %3D to =) and replaces + with space.

    # URL is http://google.com/?a=b&c=hello+%3D+hi
urlParam()
#=> {'a': 'b', 'c': 'hello = hi'}

    Returns:

    • (Object) — an object representation of the querystring parameters
